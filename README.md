# Fariha Ahrar — Portfolio Website

A responsive one-page portfolio website created with HTML and CSS. It introduces my work, provides project links, and includes a contact form.

## Live Website

[View the live portfolio](https://fariha729.github.io/Portfolio-Website/)

## Features

- Responsive layout for desktop and mobile
- Navigation links to Home, About, Projects, GitHub, and Contact sections
- Project cards that link to my GitHub work
- Contact form with a confirmation message
- CSS transitions, animations, and reduced-motion support

## Built With

- HTML5
- CSS3
